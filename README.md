#  Simple Static Website

A basic multi-page static website built using **HTML** and **CSS**.  
This project is created to understand website structure, navigation, styling, and responsive design.

---

##  Pages Included

- **Home Page** (`index.html`)
- **About Us Page** (`about.html`)
- **Contact Us Page** (`contact.html`)

Each page is connected using a common navigation bar.

---

##  Technologies Used

- HTML5
- CSS3
- Media Queries (for responsiveness)

---

##  Features

- Clean and simple UI
- Responsive navigation bar
- Multi-page website structure
- Contact form (UI only)
- Footer section on every page
- Mobile-friendly design
